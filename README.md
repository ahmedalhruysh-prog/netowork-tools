# Network Tools App

 

تطبيق *Network Tools* عبارة عن مجموعة أدوات شبكية لتسهيل اختبار الشبكة واستكشافها، وتشمل:  
- Ping  
- Traceroute  
- Port Scanner  
- Whois Lookup  

---

## 📦 المتطلبات

- Flutter SDK مثبت على جهازك (يفضل الإصدار >= 8.0.12)  
- Dart SDK (يأتي مع Flutter)  
-  هاتف حقيقي لتشغيل التطبيق




تنزيل المكتب دخل ملف حpubspec.yaml


dependencies:

  flutter:

    sdk: flutter
  dart_ping: ^9.0.0        # لإجراء Ping وTraceroute

  network_tools: ^3.0.2    # لفحص المنافذ
  
  whois: ^1.8.0             # لاستعلام معلومات النطاقات
